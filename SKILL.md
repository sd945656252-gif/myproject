# AI Creative Hub

Commercial-grade AI creative workstation with intelligent routing for image, video, and audio generation.

## Features

- **Image Generation**: Text-to-image, image-to-image, inpainting, ControlNet
- **Video Generation**: Text-to-video, image-to-video, video-to-video, upscaling
- **Audio Generation**: Music generation, text-to-speech
- **Prompt Engineering**: Image-to-text, prompt optimization
- **Workflows**: Automated six-step video creation pipeline
- **History**: Complete task history and statistics

## Quick Start

```bash
# Clone and setup
git clone <repository>
cd ai-creative-hub
cp .env.example .env

# Start services
./start.sh

# Access
# Frontend: http://localhost:3000
# Backend API: http://localhost:8000
# API Docs: http://localhost:8000/docs
```
